# Arduino-Based V6 Engine Simulation

## Overview
This project demonstrates a simple V6 engine simulation using Arduino Uno and a geared DC motor.

## Components Used
- Arduino Uno
- Geared DC Motor
- DC Speed Controller
- 12V Battery

## Features
- Simulates engine movement
- Motor speed control
- Basic embedded systems project

## Future Improvements
- Laptop communication
- Cylinder deactivation
- Automation system
